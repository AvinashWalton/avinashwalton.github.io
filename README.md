# Avinash Walton — Official Portfolio

> Personal portfolio website of **Avinash Walton (Sonu Kumar Suman)** — Indian Creative Entrepreneur, Data Analyst, Podcaster, and Founder of Indraseva Group.

🌐 **Live Site:** [avinashwalton.github.io](https://avinashwalton.github.io)

---

## About This Project

This is my official portfolio website, hosted on GitHub Pages. It showcases my skills, certifications, projects, verified works, published books, podcast, and contact information — all in one place.

The design is inspired by my Blogger theme and rebuilt from scratch as a fully static site using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## Features

- ✨ Animated cosmic starfield background (Vedic / Shatabhisha theme)
- 🖱️ Custom dual-layer cursor with smooth ring animation
- ⌨️ Typewriter effect with multiple rotating roles
- 📊 Animated skill bars triggered on scroll
- 🔢 Animated stat counters on hero section
- 📜 Rotating divine shloka section (Bhagavad Gita & Upanishads)
- 🕉️ Floating Vedic mantras (ॐ, सीताराम, राधे कृष्ण)
- 📚 Auto-scrolling books carousel with pause-on-hover
- 🎙️ Spotify podcast embed
- 📬 Contact form via FormSubmit (no backend needed)
- 🔝 Scroll progress bar + Back to Top button
- 📱 Fully responsive for mobile and desktop

| Technology | Usage |
